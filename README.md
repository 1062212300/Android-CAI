# Android-CAI

##  项目简介

一款“美食App”——让你吃到嗨！

全国美食精品荟萃，集聚八大菜系，川鲁粤苏浙闽湘徽，从家常菜到人气料理，从正餐到甜点，从满汉全席到养生汤。搜索美食智慧，品评美食文化，体会美食乐趣。菜，简单的名字，幸福的滋味！海量不同菜系菜谱，满足您的个性化需求

风味民肴，特色食材，独家匠艺！各种美食，随心所享！让你体验前所未有的爆炸口感！
